# Data Analysis

Il dataset contiene più di 2 milioni di ricette, ciascuna delle quali è descritta da 7 colonne:<br>

|    Nome    |                  Descrizione                  |
|    :--     |                      :--                      |
| Unnamed: 0 | intero che identifica univocamente la ricetta |
| title      | titolo della ricetta                          |
| ingredients| dosi degli ingredienti                        |
| directions | istruzioni della ricetta                      |
| link       | url della ricetta originale                   |
| source     | fonte di provenienza della ricetta            |
| NER        | lista degli ingredienti                       |
